# ZillowGuessr - The Zillow Listing Price Guessing Game

This is a work in progress web application hoping to be accessible to the public soon but serves currently as a project for me to learn React Native w/ Flask on the backend to perform the data-scraping job

## How to run

This project is Dockerized meaning all of the services are ran with a simple command:
`docker-compose up`

This will get everything up and running to begin using the app in Development mode, be sure to add the `--build` flag initially to build all the dependencies!
